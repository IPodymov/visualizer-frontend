# Функциональность Проекта

Приложение "Academic Plan Visualizer" предоставляет инструменты для анализа и визуализации учебных планов ВУЗов.

## Основные возможности

### 1. Просмотр Учебных Планов
*   **Список планов**: На главной странице отображается сетка доступных учебных планов с возможностью фильтрации по году поступления и поиска по специальности.
*   **Карточка плана**: Краткая информация о плане (специальность, год, количество часов, семестров).
*   **Детальная страница**: Полная информация о плане, включая список дисциплин. Доступны различные представления: табличное (`PlanTable`) и в виде карты/графа (`PlanMap`).

### 2. Сравнение Планов (`Compare Page`)
*   **Выбор планов**: Пользователь может выбрать планы на главной странице (до 3-х), отметив их галочкой "Сравнить".
*   Появляется плавающая кнопка (FAB) для перехода к сравнению.
*   **Визуализация**: На странице сравнения отображаются виджеты "Вычислительный ландшафт" (Computing Landscape) для выбранных планов бок о бок, а также объединенный график (Unified Landscape Chart), что позволяет наглядно оценить различия в нагрузке и структуре.

### 3. Личный Кабинет и Пользовательские Предпочтения
*   **Регистрация и Авторизация**: Вход в систему для доступа к персонализированным функциям.
*   **Профиль пользователя**: Страница (`Profile Page`) для управления своими данными и просмотра сохраненных элементов.
*   **Избранное**: Возможность добавлять интересующие учебные планы в список избранных для быстрого доступа.
*   Специализированный виджет, отображающий распределение нагрузки по семестрам и типам дисциплин.
*   Позволяет быстро оценить сложность и фокус учебной программы.

### 5. Экспорт
*   Возможность экспорта данных (реализована базовая поддержка через библиотеки `html2canvas` и `jspdf` для создания отчетов).

## UI/UX Решения
*   **Адаптивная верстка**: Интерфейс корректно отображается на мобильных устройствах и десктопах.
*   **Интерактивные элементы**: Анимации при наведении, плавные переходы, удобная навигация.
*   **Цветовая схема**: Использование акцентных цветов для выявления важных элементов интерфейса.
